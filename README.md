# FemScan-AI

AI-assisted cervical cell classification and endometriosis symptom risk scoring.

## Modules

| Module | Owner | Description |
|--------|-------|-------------|
| `models/cervical_classifier.py` | Person A | EfficientNetV2-S trained on SIPaKMeD (5-class) |
| `models/gradcam.py` | Person A | Grad-CAM explainability wrapper |
| `models/endo_scorer.py` | Person A | XGBoost + SHAP endometriosis risk scorer |
| `dashboard/` | Person B | Streamlit UI components |
| `app.py` | Person B | Streamlit entry point |
| `modal_deploy.py` | DevOps | Modal cloud inference endpoints |

## Output Contracts

All model output shapes are documented in [`docs/OUTPUT_CONTRACTS.md`](docs/OUTPUT_CONTRACTS.md).
Person B (UI) should code against these contracts — not against model internals.

## Quick Start

```bash
pip install -r requirements.txt

# Generate demo assets (no trained weights needed)
python scripts/prepare_demo.py

# Run the dashboard
streamlit run app.py
```

## Data

| Folder | Contents |
|--------|----------|
| `data/sipakmed/` | SIPaKMeD cervical cell dataset (not tracked — add via DVC) |
| `data/endo/` | Endometriosis symptom dataset (not tracked — add via DVC) |
| `data/demo/` | Pre-computed demo assets generated by `scripts/prepare_demo.py` |

## Cloud Deployment

```bash
modal deploy modal_deploy.py
```
